(constant) CHUNK_SIZE

This const sets the size of each chunk in bytes. This constant sets the maximum amount of data (in bytes) that each chunk can contain when sending a message in chunked form. It is calculated by taking a value and multiplying it by 1024 to convert it to kilobytes (KB), it then multiples it again by 1024 to convert it to megabytes (MB). For example, 10 * 1024 * 1024 sets the chunk size to 10 MB.
